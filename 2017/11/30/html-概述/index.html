HTTP 请求与 HTTP 响应

Content-Type：text/html

content-Encoding:”gzip” gzip 是 GNUzip 的缩写，它是一个 GNU 自由软件的文件压缩程序。它是 Jean-loupGailly 和 MarkAdler 一起开发的。最早是用于 UNIX 系统的文件压缩。我们在 Linux 中经常会用到后缀为.gz 的文件，它们就是 GZIP 格式的。现今已经成为 Internet 上使用非常普遍的一种数据压缩格式，或者说一种文件格式。

HTTP 协议上的 GZIP 编码是一种用来改进 WEB 应用程序性能的技术。大流量的 WEB 站点常常使用 GZIP 来让用户感受到更快的速度。这一般是指 WWW 服务器中安装的一个功能。当有人来访问这个服务器中的网站时，服务器中的这个功能就将网页内容压缩后从传输到来访的电脑浏览器中显示出来。一般对纯文本可压缩到原来大小的 40 % 这样传输就快了，效果就是你点击网址后会很快的显示出来，当然这也会增加服务器的复杂，一般服务器中都安装有这个功能的模块。

HTML 标记

html 的单标记元素

单标记元素，即不需要结束标记的元素。通过末尾的斜线来闭合。

tips: 对于单标记元素，尽量在 start tag 的闭合之前加上 /

  • HTML5 可以不加，但 XHTML 强制需要
  • 加上 / 可以明确表示元素结束，增加代码可读性

HTML 全局属性

有一些属性是所有的 HTML 元素都可以添加的属性。称为 HTML 全局属性

  • accesskey
  • class
  • contenteditable
  • contextmenu
  • dir
  • draggable
  • dropzone
  • hidden
  • id
  • itemid
  • itemprop
  • itemref
  • itemscope
  • itemtype
  • lang
  • spellcheck
  • style
  • tabindex
  • title
  • translate

最常用的全局属性有： id、class、style、title

title 属性通常用来给标签添加特定的文字提示，当鼠标放在该元素上时，就会弹出这个文字提示。
