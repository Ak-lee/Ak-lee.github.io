实现一个 jQuery 的 API

这里的jQuery函数，接受一个 nodeOrSelector 作为参数，既可以直接传入一个node，也可以传入一个selector选择器。参数传入后首先判断这个nodeOrSelector 究竟是node还是selector。selector用语句typeof nodeOrSelector === 'string'的if语句来判断。node用nodeOrSelector instanceof Node来判断。考虑到如果传入的是一个selector，则可能选中多个元素，所以我们使用一个伪数组来存储操作对象。为了尽量统一，即使传入的是一个节点（Node）我们也把它存在一个伪数组里面。

被操作对象存好了以后，我们就可以来添加供调用的方法。

  • addClass 方法

    addClass 接受一个参数，来设置需要添加的 class 属性。由于class 是javascript的关键字，所以务必要注意不能使用 class 为变量名， 这里使用classname变量来存储要添加的 class 属性。使用 for 循环，遍历每一个node元素，逐个用语句node[i].classList.add(classname) 来添加class属性。

  • setText 方法

    setText方法接受一个参数，来设置需要设置的文本。这里使用text来接受参数。使用 for 循环，遍历每一个node元素，逐个用语句nodes[i].textContent=text 来设置文本内容。

jQuery函数调用时，使用return语句把生成好的对象传回去。

由于 jQuery 的选择器常简写成 $, 故这里加一句window.$=jQuery;

在整个代码实现过程中，用到了闭包。我们把 nodes 对象传到了jquery函数外部。但nodes 对象内部的函数调用了jquery函数内部声明的变量。故实现了在jquery函数外部调用jquery函数内部的参数。这就是闭包的用途。
