vim 简明教程

第一级：存活

i insert 模式，按 ESC 回到 Normal 模式。

x 删当前光标所在的一个字符。

:wq 存盘 + 退出（ :w 存盘， :q 退出 ）注：:w 后可以跟文件名

dd 删除当前行，并把删除的行存到剪切板里面

p paste 粘贴剪贴板

hjkl ( 强烈推荐使用其移动光标，但不必需 ) 你也可以使用上下左右的光标。注：j 就像向下的箭头。j 向下。k 向上。h 向左。l 向右。

:help <command> 显示相关命令的帮助。你也可以就输入:help 而不跟命令。注：退出帮助需要输入 :q

在 Vim 的 Normal 模式下，所有的键就是功能键。

第二级：感觉良好

上面的那些命令只能让你存活下来，现在是时候学习一些更多的命令了，下面是我的建议：

  1. 各种插入模式

    1. a 在光标后面插入
    2. i 在光标前插入
    3. A 在行尾插入
    4. I 在行首插入
    5. o 在当前行后插入一个新行
    6. O 大写的O 。在当前行前插入一个新行
    7. cw 替换从光标所在位置后到一个单词结尾的字符

  2. 简单的移动光标

    0 数字零，到行头。

    gg 光标定位到最上面一行的第一个字符

    G 到最后一行的第一个字符

    ctrl + d 往下翻一屏

    ctrl + u 往上翻一屏

    ^ 到本行第一个不是 blank字符的位置。所谓 blank 字符就是空格，tab，换行，回车等

    $ 到本行结尾。

    g_ 到本行最后一个不是 blank 的字符。

    /pattern 搜索 pattern 的字符串。（ 如果搜索出多个匹配，可按 n 到下一个

  3. 拷贝/粘贴

    P 粘贴

    yy 拷贝当前行

  4. Undo/Redo

    u undo 撤销

    ctrl + r redo 重做

复制一行在粘贴： yy + p

删除一行：dd

删除两行： 2dd

以单词为单位进行光标移动：e 到当前单词的词尾。b 到当前单词的词首。w 到下一个词的词首。

一行内快速定位到某个字母上。 funny 这个单词，目前光标在f上。按 fy 就能定位到最后的 y 上。

修改：

ci' c 意思是 change。 i 意思是 inner 内部。 ci' 意思是change 单引号里面的内容。

vi' 选中单引号中的内容。

di' 删除单引号中的内容。

diw 删除当前的这个单词。

df空格 删除到空格的位置。空格也一起删掉。

dt空格 删除到空格的位置。空格要保留。

用 v 来选中多行，方便进行删除等操作。
