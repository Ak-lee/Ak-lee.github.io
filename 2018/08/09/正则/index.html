JS 中的正则表达式

正则表达式是强大的字符串匹配工具。是一种人类很难读懂的文字。

str.search 能找到匹配的元素所在的位置（下标index）

str.match 能把所有匹配的元素都提取出来。

正则中的转义：

/d 表示数字。

g 代表 global 全部匹配。

+ 是一个量词，若干，多少都可以，表示匹配连续的多个。

str.replace : 替换所有匹配。返回替换后的字符串。例如用于敏感词过滤。

var re = /北京|上海|广州|深圳/g 。| 竖线表示 或

方括号： [abc]pc 用于匹配 apc、bpc、cpc

方括号 表示其中任一个皆可。

范围：

[a-z] 小写字母都要。

[0-9] 数字都要。[0-9] 的作用和 \d 相同。

[^a-z] 除了小写字母，其他都要。

[^a-z0-9] 除了英文字母和数字，其他都要。

转义字符：

  • . (点) 表示任意字符
  • \d 数字 [0-9]
  • \w 英文数字下划线 [a-z0-9_ ]
  • \s 即 space 空白字符。
  • \D [^0-9]
  • \W [^a-z0-9_]
  • \S 非空白字符

正则的贪婪特性： 正则会优先匹配长的字符串。

量词

  • 量词表示出现的次数
  • {n, m} ，至少出现 n 次，最多 m 次

常用量词

  • {n, } 至少 n 次
  • * 任意次 {0，}
  • ? 零次或一次 {0,1} 可有一次可无
  • + 一次或任意次 {1，}
  • { n } 正好 n 次

str.test(reg) 只要字符串一部分符合要求就返回 true

^ 表示行首，字符串的开头。

$ 表示行尾，字符串的结尾。
